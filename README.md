# Python Crafting Interpreters

A Python (3.8 +) implementation of the **Crafting Interpreters** book by Bob Nystrom. This project includes a scanner, parser, and interpreter for the Lox language.


## Setup

```bash

1. Running the Interpreter

# From the project root, run a Lox source file:

python3 -m lox.lox lox/test.lox

2. Running Tests

pytest

# Ensure tests are located in the tests/ directory
